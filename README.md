# Calculator

A fancy calculator to do some Math!

## Description

This project was created for a EPAM internal Test

### How to use the project?

- **Step 1**
  Run the next command in your terminal

#### `npm run install`

- **Step 2**
  run the project

#### `npm run start`

- **Build**
  If you want change the code, you must recompile

#### `npm run build`
